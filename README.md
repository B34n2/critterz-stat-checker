# critterz-stat-checker
Check your Critterz stats

## Requirements
- VC++ Build Tools
- Python 3.x

## Instructions
1. Install required Python packages with `pip install -r requirements.txt`
2. Create `.env` with required variables
3. Run `bot.py`

## .ENV

```BOT_TOKEN=
BOT_PREFIX=
EMBED_COLOR=<HEX-COLOR-CODE>
```
